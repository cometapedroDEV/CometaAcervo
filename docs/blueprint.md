# **App Name**: CursoDigital

## Core Features:

- Admin Authentication: Secure login for administrators to manage courses.
- Admin Course Management: Admin dashboard to add, edit, and delete course details (title, description, price, access keys/links).
- Course Description Assistant: An AI tool to assist administrators in generating compelling course descriptions or summarizing key learning points efficiently.
- Buyer Authentication: User registration and login for buyers to access and manage their purchases.
- Course Discovery & Search: Allow buyers to browse available courses, view details, and search for specific topics.
- Course Purchase Flow: Streamlined process for buyers to select and purchase courses.
- Course Access Delivery: Automatically provide purchased course access (e.g., links or keys) to the buyer upon successful payment.

## Style Guidelines:

- Primary interactive color (buttons, highlights): A vibrant, bold orange (#F07E1F) to signify action and importance.
- Main background color: Crisp white (#FFFFFF), providing a clean and spacious feel for content visibility, as requested by the user.
- Tertiary accent color: Solid black (#000000), utilized for strong headings, essential text, and creating high-contrast elements, as requested by the user.
- Headline font: 'Poppins' (sans-serif) for a modern, geometric, and precise look for titles and main sections.
- Body font: 'Inter' (sans-serif) for clear, highly readable text in course descriptions, administrative panels, and general content.
- Utilize modern, clean, and line-art based icons that complement the digital and educational nature of the platform. Icons should provide intuitive visual cues for actions and categories.
- Employ a responsive and card-based layout for course listings, ensuring accessibility and a pleasant viewing experience across various devices. Implement clear sectioning for administrative tools and user-facing content.
- Incorporate subtle hover effects on interactive elements like buttons and course cards, along with smooth transitions for page loads and modal displays, enhancing user engagement without distraction.
